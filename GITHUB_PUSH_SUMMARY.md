## 🚀 How to Use from GitHub

### Step 1: Clone Repository
```bash
git clone https://github.com/adityafilesx/Onion-Trading.git
cd Onion-Trading
```

### Step 2: Setup Environment
```bash
# Create virtual environment
python3 -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Step 3: Configure API Keys
```bash
# Create .env file with your Binance testnet credentials
cat > .env << EOF
BINANCE_API_KEY="your_api_key_here"
BINANCE_SECRET_KEY="your_secret_key_here"
BASE_URL="https://testnet.binancefuture.com"
EOF
```

### Step 4: Run Tests
```bash
# Execute full test suite
python direct_test.py

# Or run alternative test runner
python test_execution.py
```

### Step 5: Start Trading
```bash
# Check account balance
python cli.py account-info

# Test order (dry-run, no execution)
python cli.py place-order --symbol BTCUSDT --side BUY --type MARKET --quantity 0.001 --dry-run

# Execute live order
python cli.py place-order --symbol BTCUSDT --side BUY --type MARKET --quantity 0.001

# View order history
cat logs/orders_history.json | jq '.'
```

### What's Included
✅ Production-ready code  
✅ Comprehensive tests

### What's NOT Included (By Design)
❌ `.env` file with real API credentials  
❌ Virtual environment (`/.venv/`)  
❌ IDE configuration (`.vscode/`)  
❌ OS-specific files

### Security Best Practices
```bash
# Never commit .env file
echo ".env" >> .gitignore

# Use environment variables for secrets
export BINANCE_API_KEY="your_key"
export BINANCE_SECRET_KEY="your_secret"

# Create .env.example template
cp .env .env.example
# Remove actual values from .env.example
```
