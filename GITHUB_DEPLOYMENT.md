## How to Use from GitHub

### 1. Clone the Repository
```bash
git clone https://github.com/adityafilesx/Onion-Trading.git
cd Onion-Trading
```

### 2. Set Up Environment
```bash
# Create virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

### 3. Configure API Keys
```bash
# Copy .env template and add your keys
cp .env.example .env
# Edit .env with your Binance testnet API credentials
```

### 4. Run Tests
```bash
python direct_test.py
```

### 5. Start Trading
```bash
# Check account
python cli.py account-info

# Place test order (dry-run)
python cli.py place-order --symbol BTCUSDT --side BUY --type MARKET --quantity 0.001 --dry-run

# Execute live order
python cli.py place-order --symbol BTCUSDT --side BUY --type MARKET --quantity 0.001
```
